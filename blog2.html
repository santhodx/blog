In this blog I am talking about 'Required' Attribute in HTML.
It is widely used to notify the users that the particular text box cannot be empty. Most common websites that use Required attributes are Booking/Reservation websites including hotels, airlines etc. The required attribute works with the following input types: text, search, url, tel, email, password, date pickers, number, checkbox, radio, and file.
In the code sample below I have implemented a sample form submission page that requires a username and password from the users. 'Required' attribute feature is applied to the 'Username' box, so that when a user click on the submit button leaving the textbox empty, the page stays the same and prompts user to fill the specific field.

The code sample within the body is: 

   
    body
        
        form action="" 
            Username: 
            input type="text"
                   name="usrname"
                   required> 
            br Password: 
            input type="password"
                   name="password"> 
            br 
            input type="submit"
        form 
    
    body
