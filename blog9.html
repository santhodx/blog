In this blog I am talking about one of the styling effect in CSS called 'text-shadow'.
What 'text-shadow' does is that it gives shadow to your text. In its simplest use, you only specify the horizontal shadow (2px) and the vertical shadow (2px). To add a color, type the color name as the third input following the horizontal and vertical inputs. To add multiple shadows you can add a comma-separated list of shadows.

The code snippet within the body is: 

   
    h1 {
        text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;
      }
