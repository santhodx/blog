In this blog I am talking about 'Map' Attribute in HTML.
'Map' tag helps to define image map. An image map is an image where users are able to click on specific parts on the image to get the result. It provides clickable areas.
The name attribute within the map tag is the same name used in the usemap of img.src. This establishes a connection between the map and image.

The code sample within the body is: 

   
    img src="table.jpg" alt="Table" usemap="#tablemap" width="400" height="380">

    map name="tablemap">
      area shape="rect" coords="28,60,290,322" alt="Lamp" href="lamp.htm">
      area shape="rect" coords="295,178,310,240" alt="Mug" href="mug.htm">
      area shape="circle" coords="348,286,41" alt="Pencil" href="pencil.htm">
    /map>
