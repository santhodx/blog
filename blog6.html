In this blog I am talking about 'onkeydown' Event in JavaScript.
What 'onkeydown' does is that, it fires a function when a user presses any key in the keyboard. While 'onkeypress' also does a similar job, it does not support few keys such as alt, command, control, shift etc. On the contrary, 'onkeydown' applies to every keys on a keyboard.
One real life example of the 'onkeydown' event usage is, when you try to download a file, a pop up appears saying 'press any key to initiate the download.' This can be achieved with the help of 'onkeydown' event.

The code sample within the body is: 

   
    input type="text" onkeydown="myFunc()">
    script>
        function myFunc() {
          alert("You pressed a key inside the input field");
        }
        /script>
